How to read it — and what it won't do

The box widens the space; it doesn't choose for you. A handful of options on each sub-function already describe thousands of combinations — most of them nonsense, and not every option is compatible with every other. The value is forcing the alternatives into the open so a path is chosen against the full field, not defended because it was first.

Trace a path by picking the active path, then clicking the dot on one option per row. Write the reasoning in each path's summary card. Click any text to edit.

What this is — and what it isn't

A Pugh matrix compares each concept against one baseline — the datum — criterion by criterion: better (+), same (0), or worse (−). It's deliberately coarse, because at concept stage a ± judgement you can defend beats a decimal score you can't. Weights set what matters; the weighted total ranks the options.

The real output is below the table: a sensitivity check that tells you whether the ranking survives when the weights move. A recommendation that holds up under reweighting is one you can defend upstairs. One that flips on a small change is a finding in itself — say so.

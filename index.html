On saving — read this once

Boxes autosave to this browser on this device. That's convenient, not durable: clearing site data or switching machines loses them. Treat Export / Back up all as the real archive — a JSON file you keep (in your repo, in Drive, wherever). It's also how you hand a box to a client.

To run this from GitHub: put this file in a repo as index.html, then Settings → Pages → deploy from main / root. It'll be live at your-name.github.io/repo-name/. No build step, nothing to install.

How to read it — and what it won't do

The box widens the space; it doesn't choose for you. A handful of options on each sub-function already describe thousands of combinations — most of them nonsense, and not every option is compatible with every other. The value is forcing the alternatives into the open so a path is chosen against the full field, not defended because it was first.

Trace a path by picking the active path (below), then clicking the dot on one option per row. Write the reasoning for each path in its summary card — that's what feeds the study's tradeoffs. Click any text to edit it.

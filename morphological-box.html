VG ENGINEERING Morphological box · working template

Concept generation · Design-space map

Break the problem into sub-functions. Lay out every way to solve each. Then trace the paths worth studying.

A morphological box is how the design space gets widened on purpose before it's narrowed. List the sub-functions the design has to perform down the left, the candidate solution principles for each across the row, then pick one principle per row to trace a complete concept. The 3–4 genuinely different paths you trace here are the concepts that go into the study.

How to read it — and what it won't do

The box widens the space; it doesn't choose for you. Twelve sub-functions with a handful of options each describe thousands of combinations — most of them nonsense, and not every option is compatible with every other. The value is forcing the alternatives into the open so a path is chosen against the full field, not defended because it was first.

Trace a path by selecting the active path (top), then clicking the dot on one option in each row. Edit any label by clicking the text. Use Print / Save as PDF to drop the finished box and its path summaries into the concept study.
